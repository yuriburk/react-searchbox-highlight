# react-searchbox-highlight

![SearchBox gif](https://i.imgur.com/MQJXzv5.gif)

This is a easy searchbox component ready to use with highlight and loading features.

## Usage

To install dependencies

```
$ yarn install
```

To build

```
$ yarn build
```

To run tests

```
$ yarn test
```

To run Storybook

```
$ yarn storybook
```
